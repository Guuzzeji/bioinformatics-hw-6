# needle_wun_algorithm

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run example
```

or 

```bash
bun run main.js --seq-1 SEQUENCE --seq-2 SEQUENCE
```

This project was created using `bun init` in bun v1.1.27. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
